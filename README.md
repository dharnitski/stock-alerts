## Prerequisites

* Node/NPM
* npm install -g serverless
* setup ~/.aws/credentials
* npm install -g vue-cli

## Deploy Server

    serverless deploy


## Run Clint locally 

    cd client
    npm run dev

## Build Client

    cd client
    npm run build


## Deploy Client

    serverless client deploy
